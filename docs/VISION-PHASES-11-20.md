# VortexOS — Vision: Phases 11–20

> Goal: make VortexOS the **best experimental OS** — not just a Windows-9x clone, but
> *Serious Sam's creation*: superb feel, deep functionality, distinctive design and identity.
> Phases 0–9 built the real machine (microkernel, FS, shell, multi-user, OPFS, networking).
> These ten phases make it a joy to use and unmistakably **Vortex**.

The arc: **feel → polish → settings → tray → reliability → launch → apps → games → identity → reach.**

## Phase 11 — Window Manager Mastery
The foundation of "feel." Make windows perfect:
- 8-direction edge/corner resize (not just the corner grip), with live preview.
- **Snap & tile** — drag a window to an edge to half/quarter-tile (Aero-Snap style); snap to other windows.
- **Alt+Tab** task switcher with icons/previews; keyboard window control (move/size/close/cycle).
- Smooth open / close / minimize / maximize animations, subtle window shadows, focus polish.

## Phase 12 — The Authenticity & Feel Pass
Sweat every micro-interaction until it's indistinguishable from a real OS:
- A sound for every action; hover states, button-press feedback, focus rings.
- Hourglass / busy cursors, tooltips everywhere, live status bars, drag ghosts.
- Consistent right-click context menus on every surface; double-click timing; the classic "click."

## Phase 13 — Control Panel & System Settings, Done Right
A comprehensive, authentic settings suite:
- **Mouse** (double-click speed, pointer schemes/trails), **Keyboard** (repeat rate, shortcuts).
- **Sounds** (a per-event scheme editor), **Display** (themes, resolution sim).
- **Screensavers** engine (Starfield / Mystify / Pipes) + idle activation.
- Date/Time, Regional, Accessibility, Power, and a proper **About Vortex** / System panel.

## Phase 14 — Notifications, Tray & Taskbar 2.0
- A real **system tray**: live volume slider, network/online status, clock + calendar popup.
- **Toast / balloon notifications** + a notification center; WinPopup messages as proper toasts.
- Taskbar grouping, window previews on hover, Quick Launch, "Show Desktop."

## Phase 15 — Reliability, Performance & the Honest OS
- 60fps everywhere; lazy-load apps; faster boot; **session restore**.
- Error boundaries that show an authentic **Blue Screen of Death / Dr. Watson** instead of a
  white screen — and recover gracefully.
- A real **Task Manager** with live (synthesized) CPU/memory graphs and "Not Responding."

## Phase 16 — Search, Run & Launcher
- A genuinely useful **Find** (by name / content / date, live results).
- A working **Run** dialog (apps, paths, commands); recent documents; jump lists.
- The experimental twist: a modern **command palette** — fuzzy-launch anything from the keyboard.

## Phase 17 — The Apps, Deepened (Productivity)
Make the core apps genuinely good:
- **WordPad** (rich text — bold/italic/fonts, save .rtf), a stronger Notes.
- **Calculator** (scientific + programmer modes), **Paint** (shapes, fill, zoom, undo).
- **Image Viewer** (zoom/rotate/slideshow), **Media Player** (playlists + a Web-Audio visualizer).

## Phase 18 — The Apps, Deepened (Fun & Games)
The Win9x soul:
- A real games suite — **Solitaire, FreeCell, Minesweeper (difficulties), Hearts, Snake/Tetris**.
- **Per-user high scores** synced to the backend; best-times; a little arcade.

## Phase 19 — Vortex Identity & Delight
Make it a CREATION, not a clone:
- A signature **Vortex theme**, custom boot/shutdown sequences, an original sound scheme.
- **About Vortex — by Serious Sam** with credits; a **first-run welcome tour**; tips.
- **Easter eggs** (hidden credits, a secret app, konami code), a tasteful "Vortex helper"
  assistant, and small achievements. This is the soul of the OS.

## Phase 20 — Reach: Responsive, Touch & Accessibility
Best experience for everyone, everywhere:
- Responsive / mobile layout; touch gestures (tap, long-press, drag); on-screen keyboard.
- Full keyboard navigation; screen-reader labels; high-contrast / large-text modes.
- PWA install polish so it feels native when installed.

**After 20:** Phases 21–29 remain open for whatever the OS wants next; **Phase 30** is the
big "go full" Web-Workers + compositor rebuild (design in `PHASE-30-PLAN.md`).
